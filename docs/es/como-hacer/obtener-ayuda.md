# Obtener Ayuda

> Estado verificado al **13 de marzo de 2026**.

## Vista rapida

- Complejidad: Principiante
- Tiempo tipico: 5 minutos
- Resultado: triage mas rapido y handoff de soporte consistente

## Canales

1. GitHub issues para bugs/features
2. Discusiones en PR para cambios de codigo
3. Documentacion publica y guias de troubleshooting

Si necesitas un punto de partida mas rapido, usa:

- [Ejecutar y probar](./ejecutar-y-probar.md)
- [Depuracion y troubleshooting](./depuracion-y-solucion-de-problemas.md)

## Checklist de triage

Antes de abrir issue:

- incluir comando exacto y error
- incluir modo runtime (`dev` vs `dev --native`)
- incluir plataforma/versiones
- incluir path minimo reproducible de funcion

## Template minimo

```text
Version FastFN:
Modo:
OS:
Comando:
Esperado:
Actual:
Pasos para reproducir:
```
